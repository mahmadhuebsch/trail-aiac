# Code of Conduct

This project adopts version 2.1 of the Contributor Covenant as its
code of conduct. The full text — including the standards we expect,
enforcement responsibilities, scope, the enforcement guidelines, and
the four-tier consequence ladder — is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project, contributors agree to be bound by
that document.

## Reporting

To report a concern, reach the project maintainer
[@mahmadhuebsch](https://github.com/mahmadhuebsch) on GitHub — the
profile page lists current contact options. Reports are read by the
maintainer only and handled in line with the Enforcement Guidelines
linked above.
